# Lab-Branching-and-Merging-Web-UI-
Lab: Branching and Merging (Web UI)


# Objectives
After completing this lab, you will be able to:

- Create a branch
- Commit changes to a child branch
- Open a pull request
- Merge a pull request into the main branch
